# Check if Version is Between Bounds

Check if Version is Between Bounds

## Usage

``` r
version_between(software_version, lower, greater_strict)
```

## Arguments

- software_version:

  Version string to check

- lower:

  Lower bound for version check (inclusive)

- greater_strict:

  Upper bound for version check (exclusive)

## Value

TRUE if software_version is between bounds, FALSE otherwise
