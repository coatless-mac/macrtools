# Check R Version

Check R Version

## Usage

``` r
is_r_version(target_version, compare_major_minor = TRUE)
```

## Arguments

- target_version:

  Target R version to check against (e.g., "4.0")

- compare_major_minor:

  Whether to compare only major.minor (TRUE) or major.minor.patch
  (FALSE)

## Value

TRUE if R version matches target_version, FALSE otherwise
